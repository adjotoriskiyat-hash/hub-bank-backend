# Hub Bank Backend 🏦

A secure fintech API built for the TS Academy assessment, featuring real-time identity verification and external bank transfers.

## 🚀 Features
- **Identity Verification (Requirement 1):** Integrated with the NibssByPhoenix API to validate BVN/NIN before account creation.
- **Wallet Creation (Requirement 2):** Automatically generates a secure wallet for new users with a default welcome bonus of ₦15,000.
- **Transaction Management (Requirements 3 & 4):** Supports secure internal transfers, external inter-bank transfers (Bank Code 128), and transaction history tracking.
- **External API Integration (Requirement 5):** Successfully authenticated and connected to the NibssByPhoenix fintech ecosystem.
- **Authentication:** Secure user login and registration using JWT and bcrypt.

## 🛠️ Tech Stack
- **Backend:** Node.js & Express.js
- **Database:** MongoDB & Mongoose
- **API Client:** Axios
- **Environment Management:** Dotenv
